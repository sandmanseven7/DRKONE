# EcoCalc – Outil web d’économie internationale

EcoCalc est une petite application web statique (HTML/CSS/JavaScript) destinée à illustrer des notions d’économie internationale à partir de vos chapitres et exercices.

## Contenu de l’outil

- **Module de calcul de Ricardo (Chapitre 02)**  
  - Saisie des heures de travail par unité pour deux biens (Poisson, Phosphate) et deux pays (Sénégal, Maroc).  
  - Calcul automatique :  
    - Avantage absolu (comparaison directe des heures de travail).  
    - Coûts d’opportunité de chaque bien en termes de l’autre.  
    - Proposition de spécialisation par avantage comparatif.

- **Simulateur de politique commerciale (Chapitre 03)**  
  - Sélecteur d’instrument de protectionnisme (droit de douane, quotas, contenu local, barrière administrative).  
  - Tableau explicatif : mécanisme, qui supporte le coût, exemple d’étude de cas.

- **Quiz d’auto-évaluation (Chapitres 01 & 02)**  
  - QCM basés sur les définitions générales (économie internationale) et le modèle de Ricardo / coût d’opportunité.  
  - Validation immédiate avec feedback détaillé.

## Fichiers principaux

- `index.html` : structure de la page et des trois modules.
- `style.css` : mise en forme (thème sombre, cartes, tableaux, quiz).
- `script.js` : logique de calcul de Ricardo, simulateur de politique commerciale, moteur de quiz.

## Utilisation

1. Ouvrez simplement `index.html` dans un navigateur moderne (Chrome, Firefox, Edge, Safari).  
2. Aucune installation ni dépendance supplémentaire n’est nécessaire.
